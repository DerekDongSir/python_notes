JavaScript简介

JavaScript是什么？

JavaSctipt是一门弱类型、解释性语言，由网景公司于1995年提出。主要用于构建用户交互式网页。简称JS。JS中一切皆对象。

JavaScript的重要组成部分

  • ECMAScript

  • BOM编程

  • DOM编程

JavaScript和Java的关系（了解）

JavaScript和Java的关系就像是雷峰塔和雷锋的关系，即没有关系。

因为网景与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。

JavaScript和ECMAScript的关系（了解）

网景公司为了让JavaScript有规范性，将其交给ECMA组织制定规范和标准，制定的标准就叫ECMAScript。

在页面以外建立以.js结尾的文件，将JS代码写到文件内，在像下面一样引入到页面上。

函数

函数是对可重用代码的封装。JS里面的函数是对象的一种特殊形式。

普通函数

//基本语法

function 函数名(参数列表){

    //一些语句

    return 返回值; //不返回时可以省略

}



//定义函数，完成将两个参数相加并返回的操作

function add(a,b){

    return a + b;

}

//使用函数

var c = add(2,3);

console.log(c); //5

函数参数

JS中的函数不够严谨，可以在使用时，少传入参数或者多传入参数。

function add(a,b){

    return a + b;

}

//少传入参数，此时参数a为2，参数b为undefined

var c = add(2);

console.log(c); //NaN，2 + undefined = NaN



//多传入参数，可以使用arguments来取出额外参数

function add(a,b){

    console.log("第一个参数（也就是a）",arguments[0]);

    console.log("第三个参数",arguments[2]);



    return a + b;

}

add(2,3,5);

匿名函数

匿名函数没有函数名，可以作为一个“值”使用。其他地方和普通函数一样。

//匿名函数的使用方式一，当作值来给变量赋值

var add = function(a,b){

    return a + b;

}

var c = add(2,3);

console.log(c); //5



//匿名函数的使用方式二，当做其他函数的参数传入

function test(f){

    var c = f(2,3);

    console.log(c);

}

test(function(a,b){

    return a + b;

}); //输出5

test(function(a,b){

    return a * b;

}); //输出6



//匿名函数的使用方式二，当做其他函数的返回值

function rf(){

    return function(a,b){

        return a + b;

    }

}

//此时，f是一个函数

var f = rf();

var r = f(2,3);

console.log(r); //输出5



//很多人喜欢简写

console.log(rf()(2,3)); //输出5

数组

JS里的数组是任意元素的集合。JS里面，数组也是对象。

基本使用

//存储一些数字

var arr = [1,3,5,7,9];

//由于弱类型，每个元素的类型可以不一样

var brr = ["ha",13,true];



//根据下标取出元素

console.log(arr[0]);// 1

console.log(arr[2]);// true

数组遍历

var arr = [1,3,5,7,9];



//下标遍历

for(var i = 0;i < arr.length;i ++){

    console.log(arr[i]);

}



//for...in...遍历

for(var i in arr){

    console.log(arr[i]);

}



//for...of...遍历（部分浏览器不支持）

for(var a of arr){

    console.log(a);

}

对象

JS的对象有三种常见创建方式：从空对象直接给属性赋值、类JSON方式直接创建、从构造函数实例化。

从空对象直接给属性赋值

var u = {};

//相当于下面这种写法

var u = new Object();



u.name = "lins";

u.age = 24;

u.sayHello = function(){

    console.log("你好！");

}



console.log(u);

u.sayHello();

类JSON方式直接创建

var u = {

    name:"lins",

    age:24,

    sayHello:function(){

        console.log("你好！");

    }

};



console.log(u);

u.sayHello();

从构造函数实例化（了解）

//定义构造函数

function User(name,age){

    //this是被实例化的对象，相当于下面的u

    this.name = name;

    this.age = age;

    this.sayHello = function(){

        console.log("你好！");

    };

}



//实例化对象，和Python不同，需通过关键字new实例化

var u = new User("lins",24);

console.log(u);

u.sayHello();

相等运算

JS中的”相等“有两种：相等和全等。

注意：不能对NaN使用相等运算，JS中一切都不等于NaN，包括它自己。

var a = "33";

var b = "33";

var c = 22;

var d = 22;

var e = "22";





//相等运算只判断值本身是否相等

console.log(a == b); //true

console.log(c == d); //true

console.log(c == e); //true

console.log(a == c); //false

console.log(a == e); //false



//全等运算先判断类型，类型相同再判断值

console.log(a === b); //true

console.log(c === d); //true

console.log(c === e); //false

console.log(a === c); //false

console.log(a === e); //false

a++和++a

var a = 2;

var b = 2;



console.log(a++); //2，a++的值和a的值相同

console.log(++b); //3，++b会先把b做加1运算



console.log(a); //3，上一步中a++是2，但之会把a做加1运算，a就变成了3

console.log(b); //3



console.log(++a); //4，此时的a的值已经为3了，因此加上1等于4

JS DOM操作

什么是DOM、DOM树

DOM(Document Object Model，文档对象模型)，是页面上标签在JS中的对象模型。通过这个模型，我们就可以使用JS对网页上的标签进行动态属性变化。

JS操作标签的操作都叫DOM操作，经常用到内置的document对象。

页面解析渲染时，就会实时、自动的产生一个比较大的JS对象，叫做DOM树对象。
DOM操作，就是先从DOM数对象上获取想要的DOM对象，然后改变其属性的操作。

像下面的HTML代码在渲染结束后就会产生如图所示的DOM树对象。

<html>

    <head></head>

    <body>

        <div>

            <p></p>

            <span></span>

        </div>

    </body>

</html>

}

add(3);



//尝试输出r的值，成功，结果为5

console.log(r);

    console.log("全局变量",window.a);

test(3); //输出两个结果：局部变量 3  全局变量 2

JS事件编程

JS是事件驱动语言。使用JS，可以在监听到一些用户的操作（特别常见的为点击操作）后自动执行一些代码，完成逻辑功能。

什么是事件？

事件（Event）宣告了一个时机，在这个时机出现时，就可以被一些监听代码捕捉到，进而执行一些在别的时机不会被执行的代码。

从简单的事件看事件三要素

事件三要素：事件源、事件、事件处理函数。

<!DOCTYPE html>

<html>

    <head>

        <title>页面标题</title>

    </head>

    <body>

        <div onclick="div_click()">一个普通的div</div>

    </body>

    <script>

        function div_click(){

            alert("您点击了div！");

        }

    </script>

</html>

这个例子中的事件三要素：

  • 事件源：被点击的div。

  • 事件：click（单击事件）。

  • 事件处理函数：div_click。

注意：onclick中的on表示监听，事件实际上是click。

常用的DOM事件

事件 生效元素 触发时机
click 任意DOM元素 用户单击鼠标左键
dblclick 任意DOM元素 用户双击鼠标左键
mouseover 任意DOM元素 用户鼠标移入元素
mouseout 任意DOM元素 用户鼠标移出元素
mousemove 任意DOM元素 用户鼠标在元素上移动
mousedown 任意DOM元素 用户按下鼠标按键
mouseup 任意DOM元素 用户松开鼠标按键
change 主要是表单元素 用户改变表单中内容后，焦点离开
focus 主要是表单元素 焦点进入
blur 主要是表单元素 焦点离开

停止事件冒泡

JS中的DOM事件有一个特性，会自动向父元素冒泡，直到冒泡到根标签html。这些父元素都可以接受到指定的事件。

事件冒泡效果演示：

<!DOCTYPE html>

<html onclick="html_click()">

    <head>

        <title>页面标题</title>

    </head>

    <body onclick="body_click()">

        <div onclick="divf_click()">

            父div

            <div onclick="divc_click()">一个普通的div</div>

        </div>

        <script>

            function divc_click(){

                console.log("您点击了普通的div！");

            }

            function divf_click(){

                console.log("您点击了父div！");

            }

            function body_click(){

                console.log("您点击了body");

            }

            function html_click(){

                console.log("您点击了html");

            }

        </script>

    </body>

</html>

如果我们想让点击子元素和点击父元素时执行不同的代码，就需要在子元素的点击事件处理函数里阻止事件冒泡。

<!DOCTYPE html>

<html onclick="html_click()">

    <head>

        <meta charset="utf-8">

        <title>页面标题</title>

    </head>

    <body onclick="body_click()">

        <div onclick="divf_click()">

            父div

            <div onclick="divc_click()">一个普通的div</div>

        </div>

        <script>

            function divc_click(){

                //使用事件处理函数的内置参数event来停止事件冒泡

                event.stopPropagation();

                console.log("您点击了普通的div！");

            }

            function divf_click(){

                //由于这里没有停止冒泡，所以点击父div时仍然冒泡

                console.log("您点击了父div！");

            }

            function body_click(){

                console.log("您点击了body");

            }

            function html_click(){

                console.log("您点击了html");

            }

        </script>

    </body>

</html>

阻止事件默认行为

JS中的DOM事件不但可以被我们监听来处理，在一些特殊情况下也有一些自己的默认行为，比如表单的提交按钮和重置按钮的点击事件就有默认行为。

<!DOCTYPE html>

<html lang="zh-cn">

<head>

    <meta charset="UTF-8">

    <title>Title</title>

</head>

<body>

    <form action="#">

        <input type="text" name="username"><br>

        <input type="submit" onclick="form_submit()">

        <input type="reset" onclick="form_reset()">

    </form>

    <script>

        function form_submit() {

            //使用事件处理函数的内置参数event来阻止默认行为

            //点击提交后，表单并不会提交

            event.preventDefault();

        }

        function form_reset() {

            //点击重置后，表单并不会重置

            event.preventDefault();

        }

    </script>

</body>

</html>

事件处理函数中获得事件源DOM（了解）

事件的处理函数里面可以直接获得事件源的DOM对象，但是在不同的浏览器中有不同的方式。

  • 在某些浏览器中，事件处理函数内置event参数，获得事件对象本身。另外一些使用window.event来做同样的事情。

  • 在某些浏览器中通过事件对象的srcElement属性获得事件源。另外一些使用事件对象的target属性来做同样的事情。

<script>

    function div_click(){

        //兼容几乎所有的浏览器获取事件对象

        var event = event ? event : window.event;

        //兼容几乎所有的浏览器获取事件源 

        var el = event.srcElement ? event.srcElement : event.target;



        alert("您点击的div的id是" + el.id); 

    }

</script>
