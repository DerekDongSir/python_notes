服务器与客户端（了解）

什么是服务器？客户端？C/S模式？

服务器 是一台专门用来提供服务的机器。

我们平时在使用QQ、微信时，是连接到了腾讯提供的服务器，并且借助它来作为中转站向其他用户发送消息。
我们在进行网络游戏时，会实时刻刻的将自己所处的位置、所释放的技能等通过服务器发送给其他玩家，来保证游戏的进行。

用户在访问服务器时，都需要使用专门的 客户端 进行连接。

上面提到的QQ、微信或是某个装在我们电脑上的某个网络游戏，都是客户端。

C/S模式 中的C指的是客户端，S指的是服务器。C/S模式，就是指使用专门的客户端来和服务器进行通信的方式。

什么是B/S模式？有什么好处？

B/S 中的B指的是浏览器。B/S模式，就是指借助浏览器来和服务器进行通信的方式。

我们平常进行的百度搜索，或者在天猫、淘宝、京东上进行购物的过程都是在借助浏览器访问百度等公司的服务器。

C/S 和 B/S 模式的比较

  C/S模式 B/S模式
出现时期 网络出现不久 互联网兴起后
初次使用方式 下载专门客户端 打开网页
发生功能或界面变更时 重新下载客户端 无需进行任何操作
对用户本地资源的占用 一般较大 一般较小
对用户的网络需求 一般较小 一般较大

通过以上比较也可以看出，B/S模式除了对网络需求较大之外，比起传统C/S模式有更多的优点。另外，随着现代网络带宽的一次次大幅提升，B/S应用在网络方面受到的限制也越来越少。

在B/S模式中，服务器除了要存放一些数据之外，还需要存放一些网页和其他资源。

超文本标记语言（HTML）

什么是HTML？

HTML ，即HyperText Markup Language（超文本标记语言）。我们平时所说的网页，即为使用HTML语言来书写的文件，也叫做超文本文档。

如何创建HTML？

HTML的本质是一个文本文件，我们只需要建立一个以html或htm结尾的文件，然后在里面书写一些内容，就能通过浏览器来展示这些内容，这就已经创建了最简单的HTML。

HTML标签

HTML是超文本标记语言，可以用标记来实现一些普通文本实现不了的操作（超文本操作），这里所说的标记，主要是指标签。

HTML标签，一般由开始标签、标签体、结束标签和标签属性组成。

关于标签的一点说明：
部分标签没有标签体和结束标签，HTML属性都有其默认值，因此也不是必须的部分。另外，由于HTML标签的不规范性，有些标签可以有多种书写方式。按照规范要求，HTML标签采用小写字母书写。HTML的属性部分可以使用双引号，也可以使用单引号。

一个最基本的HTML结构如下：

<!doctype html>

<html>

<head>

    <title>网页标题</title>

</head>

<body>

    网页主体（网页正文）

</body>

</html>

HTML的运行过程

先记住一句话：HTML是自上而下解析的。

文档类型声明

HTML语言的发展经历了一个很长的过程，为了帮助浏览器正确的解析HTML文件，每个HTML文件都需要声明自己的文档类型。

HTML5文档类型（目前最常用的文档类型）

<!DOCTYPE html>

HTML4.01严格类型（不支持过期标签和框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML4.01过渡类型（支持过期标签，不支持框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

HTML4.01框架集类型（支持过期标签和框架集）

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

HTML注释

在HTML文件中，使用<!-- -->来添加注释。如：

<!-- 单行注释 -->

<!-- 多行

注释 -->

特别注意：HTML注释只是让其中的内容不会被浏览器渲染，但仍可以具有其他功能。

网页头部

网页头部指的是<head></head>中的部分，这一部分会首先被浏览器加载，一般用于书写一些设置性的内容。

<meta>标签

一般需要写在<head></head>中，是网页的元信息。

指定网页为UTF-8编码（兼容性较好）：

<meta http-equiv="content-type" content="text/html;charset=utf-8">

指定网页为UTF-8编码（HTML5专属）：

<meta charset="utf-8">

告知IE浏览器以最接近现代浏览器的方式渲染此页面（了解即可）：

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">

自动每1秒刷新一次（了解即可）：

<meta http-equiv="refresh" content="1">

<title></title>标签

当前网页的标题，会出现在浏览器的标题栏/标签栏。也会作为将当前网页添加到收藏夹/书签时使用的默认名称。

按照HTML规范，<title></title>是唯一需要包含到<head></head>中的标签。

网页主体

网页主体指的是<body></body>中的部分，一般用于书写一些需要给用户展示的内容。

字符实体

向下面这样，给书写在body的内容添加一些空格。

<body>

    这 是一些内     容。

</body>

使用浏览器打开这个文件，可以发现：“这”和“是”中间的单个空格是正常显示的，而“内”和“容”之间的多个空格会显示成只有一个空格。这是由于浏览器会自动截短大于一个的空格，目的是为了让程序员更精确的控制文本格式。

如果想在HTML中使用不会被截短的空格，要使用字符实体&nbsp;表示。

字符实体 显示结果 描述
&nbsp;   空格
&lt; < 小于号
&gt; > 大于号
&amp; & 和号
&yen; ¥ 元
&copy; © 版权（copyright）

<br>和<hr>

像多个空格一样，直接出现在HTML中的回车换行也会被截短成一个空格。如果想在HTML中使用不会被截短的换行，要使用标签<br>代替。

如果添加一个占据一行的水平线，可以使用<hr>。

图片（重点）

在网页中，可以使用<img>标签来显示一个图像。
对于一个图像，常见的属性有三个：

src

图像的来源，可以是一个地址，也可以是一段base64码。

<!--可以是一个网络地址-->

<img src="https://www.baidu.com/img/bd_logo1.png?qua=high">



<!--可以是一个本地地址（一般采用相对路径）-->

<img src="../image/logo.png">



<!--可以是一段base64码（base64码过于冗长，这里仅为示例，无法正常显示）-->

<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAhwAAAEC...">

在线将图片转换为base64码

alt

图像的代替文字，当图像无法正确加载时，用于提示用户图片内容的文字；推荐给所有图片都加入。

<img src="../image/logo1.png" alt="百度一下，你就知道！">

title（标签通用属性）

不仅是图像，所有的标签都可以添加这个属性用于增加一个悬浮提示。

<img src="../image/logo.png" title="百度一下，你就知道！">

width和height

使用HTML属性以px为单位指定图片的宽和高，墙裂推荐为所有图片都手动指定宽高。

<img src="../image/flower.jpg" width="200px" height="200px">

是一个已经被废弃，墙裂不建议使用的标签，相关需求请使用CSS代替。

<font color="red" face="楷体" size="5">这是一段颜色为红色，字号为5的楷体文字。</font>

这是一段颜色为红色，字号为5的楷体文字。

块级元素、行内元素

块级元素

  1. 总是从一个新行开始。

  2. 总是独自占据一行。

  3. 内部可以有其他任何元素或文字。

浏览器默认行为下常见的块级元素：div、h1~h6、hr、p、table、form。

行内元素

  1. 可以和其他行内元素占同一行。

  2. 在可以有标签体时，内部只能有其他行内元素或文字。

浏览器默认行为下常见的行内元素：span、img、font、a、b、i、sub、sup、input、button。

超级链接（重点）

超级链接用来定义一个访问其他超文本文档（某一部分）的快捷方式，用户可以单击以跳转。必须有合适的href属性才能被浏览器承认为超级链接。

跳转到其他网页

<a href="网络地址或本地地址">链接描述</a>



<a herf="http://www.baidu.com/">打开百度</a>



<a herf="http://www.baidu.com/"><img src="../image/logo.png"></a>

target属性

指定页面在何处打开（不要漏写下划线）：
_blank：在新窗口打开。
_self：在当前窗口打开。

跳转到网页某一部分

可以使用链接直接跳转到网页某一部分，这叫做锚点跳转。

声明锚点位置的三种方式：

  1. 任何一个标签的id属性（注意：id属性标签唯一）。

  2. a标签的name属性。

  3. 页面顶部（无需声明，固定为#top）。

跳转到锚点：

<!--跳转到id为bt1的标签或者name为bt1的a标签-->

<a href="#bt1">跳转到标题1</a> 



<!--跳转到其他页面的锚点-->

<a href="test.html#bt2">跳转到test.html页面的标题2位置</a>



<!--跳转到页面顶部，无需声明锚点-->

<a href="#top">回到顶部</a>

表格（重点）

表格在网页刚刚出现不久时主要作为一种布局方式存在。目前主要用来显示数据。

表格不再是单一标签，而是由多个标签组成的组合效果。

<table></table>标签为表格域，表格的其他部分必须在table标签以内。
<tr></tr>标签为表格的行。

表格里面显示的数据或其他元素必须写在td或者th以内。

表单（特别重点）

表单用来提供一些可以由用户选择或输入的元素，主要用于收集用户数据提交给服务器，比如登陆注册。

表单由一个form标签和若干个其他表单元素标签组成（其他表单元素可以不依赖form单独使用）。用户在填写其他表单元素后，可以提交整个表单中name属性不为空的表单元素中的数据到服务器。

<form></form>标签

action属性规定表单提交的地址。
method属性规定表单提交的方式（取值可以为get或post）。

<input>元素（重要）

最常见的表单元素，根据其type属性的不同取值，可以显示为不同的输入组件。 placeholder属性用于在用户输入前提示用户。

表格的value属性是最终提交到服务器时数据的获取位置，可以提前给予默认值。

<form>

    <!--普通文本输入框-->

    <input name="uname" type="text" placeholder="用户名"><br>

    <!--密码输入框-->

    <input name="upass" type="password" placeholder="密码"><br>



    性别：

    <!--单选框，name属性相同的为同组单选框，checked属性指定默认选中-->

    <input name="gender" type="radio" value="m" checked="checked">男

    <input name="gender" type="radio" value="f">女<br>



    爱好：

    <!--多选框，name属性相同的为同组多选框，checked属性指定默认选中-->

    <input name="hobby" type="checkbox" value="fb">足球

    <input name="hobby" type="checkbox" value="bb">篮球

    <input name="hobby" type="checkbox" value="pp" checked="checked">乒乓球<br>



    出生日期：

    <!--日期选择器，有部分浏览器不支持-->

    <input name="birth" type="date"><br>



    喜欢的颜色：

    <!--颜色选择器，有部分浏览器不支持-->

    <input name="color" type="color"><br>



    头像：

    <!--文件上传组件，有部分浏览器不支持-->

    <input name="head" type="file"><br>



    <!--提交按钮，不要写name属性，可以不写value属性-->

    <input type="submit" value="注册">

    <!--重置按钮，不要写name属性，可以不写value属性-->

    <!--普通按钮，不要写name属性，使用value属性表示按钮上的文字-->

    <input type="button" value="按钮">

<button></button>标签

在网页上显示为一个按钮。和input标签中的几个按钮不一样的是，按钮上显示的内容由这个标签的标签体决定，因此可以组合任意行内元素使用，也可以较为自由的指定样式（之后学习）。

type

常见取值如下：

<!--一个没有特殊功能的按钮-->

<!--一个具有提交功能的按钮-->

<button type="submit">按钮</button>

<!--一个具有重置功能的按钮-->

<button type="reset">按钮</button>

特别注意：当button处在一个表单内部时，如果不手动指定该button的type属性，不同浏览器可能会将其当做普通按钮或者提交按钮。因此，这时必须为这个标签指定type属性。
