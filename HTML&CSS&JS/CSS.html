CSS介绍

什么是CSS？

CSS ，即Cascading Style Sheets（层叠样式表）。可以用来为HTML或者XML添加样式（主要是HTML）。

元素是能够指定样式的最小单位，元素主要指标签。

CSS有什么作用？

仅仅通过HTML语言来改变标签显示样式的话，只能通过HTML属性进行，这有着很多限制和困难：

  1. HTML标签提供的有关样式的属性较少。

  2. 相似的样式在不同的标签里可能通过不同的属性进行定义，较难记忆。

  3. 属性的取值不够严谨规范，浏览器表现差异明显。

  4. 需要为每个标签都进行属性指定，标签较多时实现繁琐。

使用CSS就可以解决以上的问题，更加自由和方便的定义标签样式。

CSS基本使用

style属性

每个标签都可以添加style属性，可以通过这个属性为当前标签添加样式。

<div>这是第一个div中的文字</div>

<div style="font-size:24px;">这是第二个div中的文字</div>

打开这个网页，可以明显看出第二个div中的文字较大。

写在标签style属性里的内容叫做样式表，它的基本语法格式是属性名a:属性值a;属性名b:属性值b;（如果想要添加注释，只能使用/*注释*/的方式），每对属性之间必须有;进行分隔。

任何一个标签都可以添加CSS样式，为了统一起见，之后的演示一般采用div或者span进行。

CSS的推荐使用方式

通过标签的style属性指定CSS样式的方式和标签本身的耦合性较强，不推荐使用。

更加常见的指定CSS样式的方式是通过选择器选出标签后，通过样式表来添加样式。这种方式可以在两个位置写，一个是style标签中，一个是css文件中。

<style></style>

<head>

    <style>

        选择器{

            font-size:24px;

            color:red;

            font-family:"楷体","KaiTi";

        }

    </style>

</head>

.css文件与<link>

建立一个以css为扩展名的文本文件，在里面键入以下内容：

选择器{

    font-size:24px;

    color:red;

    font-family:"楷体","KaiTi";

}

然后就可以使用<link>标签引入到HTML的head标签内。

<head>

    <link rel="stylesheet" href="css文件路径">

</head>

伪类选择器

伪类选择器是指当一些标签符合某些条件时会被选中的选择器，相当于这些标签有了这些class，但是程序员不必在标签上手动添加这个（些）class，伪类选择器必须配合其他选择器使用。

伪类主要有以下几种：

  1. :focus 选中焦点所在的标签（主要是表单元素）。

  2. :hover 选中鼠标悬浮上去的标签。（最常用）

  3. :visited 选中被浏览过的超链接。

  4. :first-of-type 选中另一个选择器选出的标签的第一个。

  5. :last-of-type 选中另一个选择器选出的标签的最后一个。

提示：chrome的开发者工具可以强行给元素加上伪类，方便调试。

全选择器（了解）

使用*（星号）选择所有元素，可配合其他选择器使用。

CSS常用优先级规则

1. style属性>style标签=link标签
2. id选择器>类选择器>标签选择器
3. 写在后面的样式表>写在前面的样式表（和第2条规则冲突时以第2条为准）
4. 全选择器是所有选择器里面优先级最低的
5. 使用在属性值后面加一个 !important 可以强制优先级最高，都带有!important时继续按照上述规则
6. 任何由程序员指定的样式优先级都大于浏览器默认样式

简单记法：
!important > style属性 > id > class > 标签 > * > 浏览器

!important写法：

<style>

    div{

        font-size:20px;

        color:red !important;

    }

</style>

属性值 说明

行高，指定内部每行文字或行内元素占据的高度。

常见用法：当盒子内部只有一行内容时，行高和高度相等能让内容垂直居中。

页面流（了解）

页面上的元素被解析后的一种渲染方式，类似于水流的方式。页面自上而下解析，遇到块级元素就让它占据一个新行，遇到行内元素就按照父元素的text-align属性按照一定的方向进行流动，如此循环进行页面渲染。

float

指定一个元素在父盒子中是临近左边还是右边。它允许周围其他文字和行内元素围绕自己进行布局。

属性值 说明
none 不浮动，默认值
left 向左浮动
right 向右浮动

当一个元素浮动时（float不为none时）不再占据块级元素的空间。
当一个行内元素浮动后，就可以为其指定宽高了。

clear

可以让一个元素主动向下移动来保持自己和其他浮动元素和自己不在同一行。

属性值 说明
none 不清除浮动，默认值
left 清除左边的浮动
right 清除右边的浮动
both 清除左右的浮动，常用

position

指定元素的定位方式，默认为static，根据页面流进行定位。当一个元素定位不为static时，可以用top、left、right、bottom和z-index来操作元素的位置。

position:relative

指定元素为相对定位，元素以在页面中原来占据的位置为原点，根据（top、left、right和bottom）的取值进行偏移。但在偏移后，原位置任然占据空间。

position:absolute

指定元素为绝对定位，元素不再占据任何页面空间。此时会根据其第一个不是static定位的父元素结合（top、left、right和bottom）进行定位。如果找不到这种父元素，则根据整个页面进行定位。

外相对，内绝对（了解）

一种在处理某些定位问题时较为常见的书写方式。当我们想让一个子元素相对于其某个父元素位置确定，我们就可以设置这个父元素为相对定位，并把这个子元素设置后绝对定位后结合（top、left、right和bottom）进行处理。

position:fixed

指定元素为固定定位，元素不再占据任何页面空间。元素会相对于整个视口（用户可以看到的页面部分）进行定位，当页面滚动时，该元素保持固定不动。

display与块级元素、行内元素

display属性可以决定元素的显示方式。

属性值 说明
block 块级元素
inline 行内元素
none 隐藏显示
inline-block 行内块元素，可以被指定宽高的类行内元素

行内元素与块级元素的比较

  块级元素 行内元素
占据空间 总是占据一个新行 可以和其他行内元素占同一行
内部内容 可以有文字和其他任意元素 只能有其他行内元素和文字
边距 可以自由指定外边距和内边距（行内块也是） 竖直方向上的内边距表现怪异*

*行内元素在垂直方向上的内边距可以扩大盒子本身面积，但是不会对其中内容或其他盒子产生影响。

z-index（了解）

决定元素距离用户的远近。当因定位方式改变发生元素重叠时，越后面书写的元素离用户越近。当一个元素定位方式不为static时，可以用z-index通过一个整数（可以为负数）指定元素离用户远近以手动改变叠放次序。数字越大，离用户越近。

元素可以被指定宽高的情况

  1. 是块级元素。

  2. 是img或input。

  3. 是行内块元素。

  4. 是float不为none的元素。

  5. 是定位方式不为static或relative的元素。

当元素满足上述任一条件时，可以被手动指定宽高。

常用的属性组成方式：

1.仅仅指定背景色：

    background:CSS颜色; 相当于 background-color:CSS颜色;

2.仅仅指定背景图片：

    background:url(图片路径); 相当于 background-image:url(图片路径);

    background:url(图片路径) no-repeat; 相当于 background-image:url(图片路径) no-repeat;

    （no-repeat用于让背景图片不重复）

3.同时指定背景色和背景图片：

    background:CSS颜色 url(图片路径);

4.指定背景图片的位置：

    background:url(图片路径) no-repeat 水平偏移 垂直偏移;

雪碧图和CSS精灵

像下面这样，由许多小图拼接成的一大张图像就可以叫做一个雪碧图（CSS Sprite，CSS精灵，音译雪碧）。

CSS精灵（CSS Sprite）是一种使用CSS属性background来从雪碧图上抠图的技术。

例子：使用CSS精灵技术制作一个。

<style>

    #btn{

        display:inline-block;

        width: 84px;

        height: 30px;



        /*url填入雪碧图路径*/

        /*指定为no-repeat重复方式*/

        /*水平偏移和垂直偏移分别取图片左上角x坐标和y坐标的 相反数 */

        background:url(bg.png) no-repeat -142px -314px;

        /*也可以向下面以下分开写*/

        /*background:url(bg.png) no-repeat;*/

        /*background-position:-142px -314px;*/



        line-height:30px;

        text-align:center;

    }

</style>

<div id="btn">按钮</div>

使用CSS精灵的好处

我们知道，当我们访问一个网页时会向服务器发送一次请求。如果这个网页中，引入了其他文件（比如CSS、图片），那么同样需要向服务器发送请求。

服务器接收到请求后，会根据请求的地址去寻找对应文件，并将它发送给客户端，这是一个比较耗时耗力的操作。

如果使用CSS精灵，那我们从同一张雪碧图上取多次图片只需要向服务器发送一次请求。

因此，合理使用CSS精灵技术，能够减少请求次数，从而

1. 减轻服务器压力
2. 减少用户的等待时间
